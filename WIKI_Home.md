# MCLCE Archive Wiki

Welcome to the **MCLCE Archive** wiki! This is a comprehensive archive of Minecraft Legacy Console Edition and related source code leaks from 2013-2021, maintained by **M1noa**.

## What is MCLCE Archive?

This repository contains organized source code from multiple Minecraft editions:

- **Console Edition (2014)** - PS3, PS4, PS Vita, Xbox 360, Xbox One, Windows
- **Pocket Edition (2013)** - Android, iOS, Raspberry Pi, Windows x86
- **Infdev (2014)** - PC Infdev custom engine
- **Dungeons & Legends (2017-2021)** - Dungeons and Legends builds
- **Story Mode (2024)** - Steam beta branches
- **Whisper LCE** - Modified Console Edition by M1noa

# Minecraft Console Edition - Pre-compiled Windows Build

Pre-compiled Windows x86_64 build of Minecraft Console Edition.

## Contents
**Source:** `Release.zip` (gofile.io/d/4srlJI)  
**Date:** 2014  
**Platform:** Windows x86_64  
**Description:** Pre-compiled Windows version of Minecraft Console Edition. Ready to run executable.

### Files
- `Minecraft.Client.exe` - Main executable (11.9 MB)
- `Minecraft.Client.pdb` - Debug symbols (93.5 MB)
- `Minecraft.Client.ilk` - Incremental linker file
- `Minecraft.Client.pch` - Precompiled header
- `Effects.msscmp` - Effects/shaders
- `iggy_w64.dll` - Iggy UI library
- `mss64.dll` - Miles Sound System
- `windows.xbox.networking.realtimesession.dll` - Xbox Live networking
- `Common/` - Common game assets
- `Windows64/` - Windows-specific files
- `Windows64Media/` - Windows media assets
- `music/` - Game music
- `redist64/` - Redistributable dependencies

This is a complete, runnable build of the Windows version extracted from the console edition codebase.

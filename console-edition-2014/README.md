# Minecraft Console Edition (2014)

Source code for Minecraft Console Edition from 2014 leaks.

## Folders

### full-multiplatform/
**Source:** `Minecraft.zip` (gofile.io/d/RRf1hv)  
**Date:** 2014  
**Platforms:** PS3, PS4 (Orbis), PS Vita, Xbox 360, Xbox One (Durango), Windows x86_64  
**Description:** Complete multi-platform console edition source code. Contains platform-specific folders for all supported consoles.

**Platform folders:**
- `PS3/` - PlayStation 3
- `Orbis/` + `OrbisMedia/` - PlayStation 4
- `PSVita/` + `PSVitaMedia/` - PlayStation Vita
- `Xbox/` - Xbox 360
- `Durango/` + `DurangoMedia/` - Xbox One
- `Windows64/` + `Windows64Media/` - Windows 64-bit
- `Common/` - Shared code across platforms

### oct-2014-backup/
**Source:** `mc-console-oct2014.zip` (gofile.io/d/pxeEVD)  
**Date:** October 10, 2014  
**Git Hash:** 5ca1eff0124dd98f5bab76cfd2052dc8718b1c18  
**Platforms:** Same as full-multiplatform  
**Description:** Backup snapshot from October 2014. Similar structure to full-multiplatform.

### ps3-only-stripped/
**Source:** `Minecraft PS3 Edition Source Code.zip` (gofile.io/d/NIawxk)  
**Date:** 2014 (modified timestamps)  
**Platforms:** PS3 only  
**Description:** Initial leak - PS3 edition with other platforms stripped out and file timestamps modified. This was the first leak before the full multi-platform version was released.

### dec-2014-build/
**Source:** `MinecraftConsoles-141217.zip` (gofile.io/d/pxeEVD)  
**Date:** December 17, 2014  
**Platforms:** Console Edition  
**Description:** Console edition build from December 2014. Contains Visual Studio solution file.

## Leak History

1. **First leak:** Anonymous user posted "Minecraft PS3 edition source code leak" - code was real but stripped all other platforms and modified file timestamps (ps3-only-stripped/)
2. **Second leak:** Another user leaked the actual full console edition source code dated 2014 with all platforms intact (full-multiplatform/)
3. **Third leak:** Yet another user posted additional leak containing MCPE (see ../pocket-edition-2013/)
